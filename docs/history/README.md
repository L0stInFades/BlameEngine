# Historical Reports (archive)

These are **point-in-time** documents — checkpoint completion reports (`CP*`), handovers,
fix/status reports, and superseded plans. They are kept for provenance and archaeology but
are **not** current guidance and may contradict the present state of the engine.

For current documentation start at [`../README.md`](../README.md). The living substitutes
for what these files used to track are:

- Roadmap / milestones → [`../07-milestones.md`](../07-milestones.md)
- Known gaps / debt → [`../TECH_DEBT.md`](../TECH_DEBT.md)
- Architecture & module status → [`../ARCHITECTURE.md`](../ARCHITECTURE.md)
- How we work → [`../../CONTRIBUTING.md`](../../CONTRIBUTING.md) and [`../ENGINEERING_HANDBOOK.md`](../ENGINEERING_HANDBOOK.md)

Do not add new files here. New status/decision records belong in `../adr/` (decisions) or
the living docs above.
